# Setup
## (OPTIONAL) Create and activate virtual environment
Create:

        python -m venv .env
Activate:

        .env\Scripts\activate

## Install dependencies 
        pip install opencv-python numpy mediapipe

# Header folder
Within your project folder, create a folder and name it "Header"
